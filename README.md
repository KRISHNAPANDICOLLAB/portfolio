# Portfolio Website - Krishnapandi M G

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## How to Run the Website

### Method 1: Direct Browser Opening (Simplest)
1. Navigate to the Portfolio folder on your computer
2. Double-click on `index.html`
3. The website will open in your default web browser

### Method 2: Using VS Code Live Server (Recommended)
1. Open the Portfolio folder in VS Code
2. Install the "Live Server" extension if you don't have it
3. Right-click on `index.html` in VS Code
4. Select "Open with Live Server"
5. The website will open at `http://localhost:5500`

### Method 3: Using Python (If Python is installed)
1. Open Command Prompt or PowerShell in the Portfolio folder
2. Run one of these commands:
   - **Python 3:** `python -m http.server 8000`
   - **Python 2:** `python -m SimpleHTTPServer 8000`
3. Open your browser and go to `http://localhost:8000`

### Method 4: Using Node.js (If Node.js is installed)
1. Install a simple server: `npm install -g http-server`
2. Open Command Prompt or PowerShell in the Portfolio folder
3. Run: `http-server -p 8000`
4. Open your browser and go to `http://localhost:8000`
